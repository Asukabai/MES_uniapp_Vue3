# MES_uniapp_Vue3
基于 Vue 3 + uniapp+ Wot UI(https://wot-design-uni.cn/guide/quick-use.html) 实现的的企业 MES 管理平台。
支持微信小程序、H5、APP 端

# 项目结构
├── src/components/  // 组件
├── src/pages/  // 页面
├── src/store/  // 状态管理
